## Getting started

<Steps>
  <Step title="Install dependencies">
    Clone the repository and run `composer install && npm install` to install all PHP and JavaScript dependencies.
  </Step>
  <Step title="Configure the environment">
    Copy `.env.example` to `.env`, set your database credentials, and run `php artisan key:generate`.
  </Step>
  <Step title="Run migrations and seeders">
    Run `php artisan migrate` followed by `php artisan db:seed` to create the schema and seed initial roles and permissions.
  </Step>
  <Step title="Log in and explore">
    Start the development server with `composer dev`, then log in with an admin account to begin configuring degree programs, professors, and students.
  </Step>
</Steps>

<Note>
SWCE uses role-based access control. Every user is assigned one of three roles — **Admin**, **Profesor**, or **Estudiante** — and sees only the features relevant to their role.
</Note>
